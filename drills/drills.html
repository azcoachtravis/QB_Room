Red Zone Decision Period

All reps inside the 20 with compressed field. Emphasize high-percentage decisions — quick game, back shoulder, check-down. No hero balls.

11v11 or 7v7 inside -20; defense plays any coverage; coach grades each decision post-play

Setup / Equipment
3 hula hoops form pocket; 1 live pass rusher from backside; receiver downfield

Daily Mesh Drill (Mohns System)
Stage 1 MESH MECHANICS (Mohns)
⏱ 10 min 📋 All QBs 📊 Run Fake · Run Decision
QB + RB work pre-snap stance, footwork, and mesh exchange every single practice — 10 minutes, no exceptions. QB toes set at 5 yards (not heels). RB tracks off QB toe landmark. Rotate: QB → C → QB → DE → out. RB → LB → out.
Setup / Equipment
OL strip for alignment landmarks; QB, RB, 1 coach as DE; cone marks 5-yd QB depth; run both sides
Coaching Points
Spacing is everything — use the OL strip. QB's toes at 5 yds give RB a consistent track. The mesh must become muscle memory before reads are added. High reps, no live defense.
Source: Glazier — Mohns Mesh Mechanics Muscle Memory
Inside Zone Mesh Drill (1st Level Read)
Stage 2 MESH MECHANICS (Mohns)
⏱ 10 min 📋 All QBs 📊 Run Decision · Run Fake
QB opens with read-side foot straight ahead, pivots 45° on 2nd step facing read-side DE. Coach/player imitates DE. Give = QB extends and rides RB through B-gap. Keep = QB pulls and attacks inside shoulder of pitch key. RB aims at inside leg of playside guard.
Setup / Equipment
Coach or scout player acts as DE; RB runs inside zone path; OL strip for spacing; repeat both sides
Coaching Points
QB must decide at the mesh point — not after. Slow it down until the read is automatic, then speed up. The RB's aim point never changes regardless of the read.
Source: CoachTube — Mohns QB Footwork & Mesh Mechanics (RPO)
Outside Zone Mesh Drill (1st Level Read)
Stage 2 MESH MECHANICS (Mohns)
⏱ 10 min 📋 All QBs 📊 Run Decision · Run Fake
QB executes OZ footwork keeping eyes on read key. If LB attacks run play → QB sets feet and throws RPO throw to TE/slot. If LB stays → QB hands off. Coach playing LB mimics current week's opponent tendency.
Setup / Equipment
LB position player or coach at 2nd level; TE or slot at RPO catch point; OL strip; run both sides
Coaching Points
The LB is the key — not the DE. QB must keep eyes at 2nd level on OZ to make the RPO decision. If he watches the mesh he loses the RPO window.
Source: AFCA Insider — Outside Zone Mesh Drill / RPO
Power Read Mesh Drill
Stage 1 MESH MECHANICS (Mohns)
⏱ 8 min 📋 All QBs 📊 Run Decision · Run Fake
QB reads EMOL (end man on line of scrimmage). If EMOL squeezes → pull and follow guard lead. If EMOL widens or sets → give to RB. Add QB GT Counter rep: QB fakes handoff, takes counter path behind pulling guard.
Setup / Equipment
Scout DE player; pulling guard executes pull; QB and RB run footwork; add full backfield as progresses
Coaching Points
Power read is a give-read, not a keep-read. The default is give. QB pulls only when the read forces it. Sell the fake on the counter — the guard pull is the key to the defense.
Source: CoachTube — Mohns Complete Offensive System (Power Read)
RPO Quick Game Off Mesh (Mohns Quick Passing System)
Stage 3 MESH MECHANICS (Mohns)
⏱ 10 min 📋 Decision Making 📊 Run Decision · Pass Decision
Off any run action mesh, QB reads 2nd level linebacker. If LB stays in run fit → throw RPO route (stick/snag/spacing). If LB drops → hand off or pull. Work Stick, Stick/Bait, Stick/Decide, Snag, and Spacing variations as season progresses.
Setup / Equipment
3-5 receivers in RPO positions; LB reads the conflict; QB executes mesh then RPO throw; 7v7 preferred
Coaching Points
The RPO decision is made with the LB in the QB's periphery — eyes must stay downfield. If QB looks at the RPO first, he telegraphs. The mesh sells the run. The LB key triggers the throw.
Source: CoachTube — Mohns Quick Passing Game (Stick/Snag/Spacing)
Pre-Snap Stance Walk-Through (Mohns Checklist)
Stage 1 MESH MECHANICS (Mohns)
⏱ 5 min 📋 All QBs 📊 Pre-Snap Read
Coach walks through Mohns' pre-snap checklist every practice before team period: 1) Toes at 5 yds. 2) Feet shoulder-width, slight crouch. 3) Eyes on defense (safety depth first). 4) Hands ready for snap. 5) Ball in "cradle" — both hands under chin, vertical angle. 6) Call made before snap.
Setup / Equipment
No equipment needed; QB-only period; coach goes position by position through checklist verbally
Coaching Points
The checklist is the habit. Do it every day until it's automatic. Mohns runs this before every team period — consistency creates the pre-snap processing that separates good QBs from great ones.
Source: Glazier — Mohns Mesh Mechanics / Pre-Snap Stance
Stick / Snag / Spacing — Release Timing Drill
Stage 1 MESH MECHANICS (Mohns)
⏱ 10 min 📋 Passing Execution 📊 Pass Release · Pass Throw
QB takes shotgun snap, 1-step set, delivers to stick route (7-yd hitch breaking back to 5 yds) before receiver fully breaks. Ball thrown to front shoulder as receiver squares up. Progress to Snag (flat + corner + stick triangle) and Spacing (3-level flood).
Setup / Equipment
3 receivers in Stick positions (5-12 yds); QB in shotgun; coach calls coverage to force correct read
Coaching Points
Completion % improves when the ball arrives as the receiver turns — not after. Mohns rebuilt his entire quick game when completion % dropped from RPO-heavy periods. The timing throw is the foundation.

Full-Field Coverage Anticipation
Stage 4 DECISION MAKING
⏱ 12 min 📋 D or F 📊 Decision Avg (Weighted)
QB reads a live 11v11 pre-snap, delivers full SOCKS call, and executes the correct decision in under 2.8 seconds. Coach grades Pass Decision (–2 to +2) on every rep. No coaching prompts during the rep.
Setup / Equipment
11v11 live; DC mixes Cover 1/2/3/4 and blitz packages each play; QB verbalizes full pre-snap call; coach grades immediately post-snap
Coaching Points
Pass Decision = +2 (Optimal): QB anticipated the window before the receiver broke — ball was already in flight. Pass Decision = +1 (Correct): right read, right window. Pass Decision = 0: correct but late. Pass Decision = –1: poor. Pass Decision = –2: turnover-worthy. Track the distribution — what % hit 3 or 4?
Source: X&O Labs — Elite Decision Training / Full Field Reads
Zero Tolerance — Ball Security Under Pressure
Stage 4 DECISION MAKING
⏱ 12 min 📋 D or F 📊 Pass Decision (–2 rate)
Full-speed 7v7 with immediate film review on any Pass Decision = –2 rep. Coach pauses the session, pulls Hudl, and QB identifies the SOCKS stage breakdown that caused the turnover-worthy decision.
Setup / Equipment
7v7 full speed; coach flags Pass Decision = –2 immediately; rep is paused; QB goes to whiteboard and traces the breakdown to its stage root (Stage 1 count error? Stage 2 pressure ID miss? Stage 3 leverage misread?)
Coaching Points
Every Pass Decision = –2 traces to a SOCKS stage. Knowing which stage tells you exactly which drill to run next. Grade all metrics on TW plays: DEC, ACC, POCK, SA — the full picture reveals patterns.
Source: Football Pipelines — Ball Security / Turnover-Worthy Elimination
Pre-Snap Blitz ID — Hot Route Assignment
Stage 2 DECISION MAKING
⏱ 10 min 📋 D or F 📊 Pressure Decision Avg
QB reads pre-snap blitz indicators (safety depth, LB alignment, CB press) and calls the hot route before the snap. Graded Pass Decision (–2 to +2) on the hot throw: Pass Decision = +2 requires the ball pre-loaded and released at 1.8 seconds.
Setup / Equipment
OL walk-through; coach shows blitz alignment; QB IDs tell and assigns hot route pre-snap; snap; throw at 1.8s max; grade DEC immediately
Coaching Points
Pass Decision = +2: named it pre-snap, ball was already coming out. Pass Decision = +1: correct ID, good execution. Pass Decision = 0: ID was late (post-snap). Pass Decision = –1: wrong hot route. Pass Decision = –2: held ball into pressure. The pre-snap ID is what separates a 3 from a 4.
Source: Axemen Football — Blitz ID / Hot Route Decision Grading
Elite Placement — Outside Shoulder Mandate
Stage 4 PASSING EXECUTION
⏱ 10 min 📋 D or F 📊 Accuracy Avg (Pass Throw)
Every throw must hit the outside shoulder away from the defender. Coach grades Pass Throw (–2 to +2) and Ball Placement (–2 to +2) on every completion. Inside shoulder throws are graded Pass Throw = –1 maximum regardless of completion or result.
Setup / Equipment
7v7 with DB coverage on every route; coach grades placement, not completion; outside shoulder = ACC 2 or 3; inside shoulder = ACC 1 max
Coaching Points
Pass Throw = +2 (Perfect): receiver catches in stride, going away from the defender. Pass Throw = +1: catchable, routine. Pass Throw = –1: receiver must adjust or catch on wrong shoulder. YAC starts at the catch point — placement is the multiplier. An Pass Throw = +2 on a dumpoff beats an Pass Throw = –1 on a deep ball.
Source: Glazier — Ball Placement / YAC Creation Clinic
Leverage-to-Throw — CB Read to Timing Delivery
Stage 3 PASSING EXECUTION
⏱ 10 min 📋 D or F 📊 Pass Pocket
QB reads CB leverage pre-snap to identify the window, then delivers on time into that window. Graded Pass Throw (–2 to +2): a 3 requires the ball to arrive as the receiver turns, not after. Connects Stage 3 SOCKS read directly to execution grade.
Setup / Equipment
CB in off or press; QB identifies leverage pre-snap; receiver attacks the identified space; coach grades timing and placement separately
Coaching Points
Pass Throw = +2: ball arrives as the receiver turns — before the defender can close. Pass Throw = +1: catchable but late. If QB waits to see the break, he is reading the receiver, not the defender. That is Stage 1 thinking at Stage 3. Grade the timing, not just the completion.
Source: Glazier — Pre-Snap Leverage / Timing Throw Integration
Space-to-Ball — Grass Read Into YAC Delivery
Stage 3 PASSING EXECUTION
⏱ 10 min 📋 D or F 📊 Ball Placement
QB identifies open grass pre-snap and delivers to the spot where the receiver catches and turns upfield. Graded Ball Placement (–2 to +2). Creates YAC by design through placement — not by asking the receiver to do it.
Setup / Equipment
Still image pre-snap to identify grass → transition to live rep attacking that space; receiver routes into the identified grass; coach grades catch-point placement
Coaching Points
Ball Placement = +2: ball to the upfield shoulder, receiver catches going forward. Ball Placement = 0: neutral. Ball Placement = –1: inside shoulder, receiver turns back. Ball Placement = –2 on a completion means the space was identified but the delivery eliminated it. That is a Ball Placement failure, not a receiver failure.
Source: Football Drills — Space Creation / YAC Throw Placement
Controlled Chaos — Pocket Navigation Under Live Rush
Stage 4 PRESSURE PERFORMANCE
⏱ 12 min 📋 D or F 📊 Pass Pocket
Three live pass rushers at full speed. QB must step up, reset, and deliver on time. Graded Pass Pocket (–2 to +2) and Pass Throw (–2 to +2) simultaneously. Pass Pocket = +2 requires a step-up AND an accurate delivery — not just one or the other.
Setup / Equipment
3 live rushers from different alignments; 1-2 receivers downfield; QB in shotgun or under center; stopwatch on snap-to-throw; grade both metrics every rep
Coaching Points
Pass Pocket = +2: climbs up, resets feet, delivers on time. Pass Pocket = 0: held pocket, got ball out. Pass Pocket = –2: QB caused the breakdown — drifted, bailed early, or created his own pressure. Separate QB-caused (Pass Pocket = –2) from OL-caused on every rep. One is a QB grade. The other is not.
Source: Athletes Acceleration — Elite Pocket Control Training
Live RPO — 2nd Level Read + Keep Decision
Stage 4 RUN GAME
⏱ 10 min 📋 D or F 📊 Run Decision
Full-speed RPO at Stage 4. QB reads 2nd level LB for give/keep AND reads CB leverage for the RPO throw simultaneously. Graded Run Decision (–2 to +2) on keeps and Pass Decision (–2 to +2) on throws — both graded in the same rep.
Setup / Equipment
Full OL + RB; 1 LB at 2nd level unblocked; CB on RPO receiver; coach grades both decisions independently per rep; run both sides
Coaching Points
Run Decision = +2: correct give/keep before the mesh point. Dec=4 (throw): ball in the right window before the break. If the QB is deciding both simultaneously, his pre-snap scan did not resolve one of them. At Stage 4 the LB read and the CB leverage are both answered before the snap.

Film-Graded Fake Carry-Out — Did a Defender Bite?
Stage 3 RUN GAME
⏱ 10 min 📋 D or F 📊 Run Fake
Every run play includes a full fake. Film is reviewed after practice and Run Fake (–2 to +2) graded: Run Fake = +2 means a named defender's eyes or feet moved toward the fake. Film reveals what the naked eye misses.
Setup / Equipment
Full team or skeleton; every play filmed; QB carries fake 5 yards minimum; coach grades Hudl film post-practice using slow-motion; share FAKE grades with QB same day
Coaching Points
Run Fake = +2: show the clip, name the defender who bit, celebrate it. Run Fake = 0: nobody moved. Run Fake = –2: QB handed off and stopped. The defense should not know who has the ball. If they know, the fake was Run Fake = –2. Film is the only honest grade.
Source: Glazier — Film-Graded Fake / Carry-Out Accountability
Game Scenario Simulation — Situation Shapes Every Pre-Snap Call
Stage 4 SITUATIONAL AWARENESS
⏱ 12 min 📋 D or F 📊 Pre-Snap Read (+2 rate)
Coach scripts a full game scenario before each rep. QB must verbalize how the situation filters his pre-snap plan. Graded Pre-Snap Read = +2 (Elite) only if the situation correctly changes his read hierarchy, clock, or target before the snap.
Setup / Equipment
Coach reads scenario aloud (score, clock, field, hash, down-distance); QB states what the situation tells him before calling the play; run 7v7; grade Pre-Snap Read on each rep
Coaching Points
Pre-Snap Read = +2: named the situation AND adjusted correctly. Pre-Snap Read = 0: adequate, nothing explicit. Pre-Snap Read = –2: situation not factored in at all. The scenario card is the trigger. If the QB ignores it, grade drops to 0 regardless of the throw. At Stage 4, every snap has a situational filter.
Source: Steve Clarkson — Situational Football / Elite QB Development
Hash + Leverage Combined Read
Stage 3 SITUATIONAL AWARENESS
⏱ 10 min 📋 D or F 📊 Pre-Snap Read avg
QB identifies the hash, reads boundary CB leverage, and connects it to situational context (down and distance) before the snap. Pre-Snap Read = +2 requires all three: hash named, leverage correct, situation applied.
Setup / Equipment
Run same play from left hash, middle, and right hash; QB calls hash AND leverage AND situation before each snap; compare decisions across all three positions
Coaching Points
Pre-Snap Read = +2: "Left hash, boundary CB inside, 3rd and 3 — slant is the gift." All three correct. Pre-Snap Read = 0: read the leverage, missed the situational filter. Pre-Snap Read = –2: no verbalization or wrong shell ID. Hash awareness is the last thing QBs learn and the first thing that wins late-game drives.

Full-Speed Mohns RPO — Live Defense
Stage 4 MESH MECHANICS (Mohns)
⏱ 12 min 📋 D or F 📊 Run Decision · Pass Decision
Complete Mohns system at full speed vs. live defense. All Stick, Snag, and Spacing variations vs. any coverage. QB runs full pre-snap checklist, executes mesh, reads 2nd level LB, throws or keeps. Graded Pass Decision, Run Fake, and Pass Throw every rep.
Setup / Equipment
Full 7v7 or 11v11; defense plays any coverage; QB runs full Mohns pre-snap checklist before every snap; coach grades all three metrics simultaneously per rep
Coaching Points
At Stage 4 the Mohns system is a reflex, not a process. If the QB is thinking about footwork, the reads are not automatic. Full speed reveals what is missing. Slow down only to fix — return to full speed immediately. Track DEC, FAKE, and ACC distributions over the period.
