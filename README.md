## The SOCKS System

**SOCKS** is a universal pre-snap operating framework that teaches quarterbacks to *anticipate* rather than react.

| Letter | Stands For | What It Tells You |
|--------|-----------|-------------------|
| **S** | Safeties | MOFO / MOFC — shell structure |
| **O** | Outside Corners | Leverage & alignment |
| **C** | Count | Box numbers → who is the key |
| **K** | Key | Who to read |
| **S** | Space | Grass reads → where to attack |

**Two Core Principles:**
- *Numbers tell you **who** to read*
- *Leverage tells you **where** to throw*
- *Pressure overrides both*

**4 Progressive Stages** (mastery-based, not grade-based):

| Stage | Focus | Typical Grade |
|-------|-------|--------------|
| Stage 1 | Count + Key | Freshman |
| Stage 2 | Safeties + Shell | Sophomore |
| Stage 3 | Outside Corners + Space | Junior |
| Stage 4 | Full SOCKS + Anticipation | Senior |
