Workflow
1. Import Hudl Game → enter Game ID, comp level, Hudl URL

3. Move Graded Plays → converts labels to scores
4. Build Dashboard → enter jersey # or ALL

7 Graded Categories
Pass Decision · Pass Pressure · Pass Pocket · Pass Release · Pass Throw · Run Decision · Run Fake · QB Impact

Each graded –2 (Poor) to +2 (Elite). Zero = expected baseline winning football.

Grading Scale — –2 to +2
+2
Elite
Significantly above expectation. Game-changing. Rare — use sparingly. Anticipation throw, perfect pocket movement, or held a linebacker with a fake.
+1
Above Expected
Clearly above baseline. QB did more than required. Correct read above the expected difficulty.
0
Expected
Baseline winning football. Most plays should be 0. Not average — correct. Grade process, not result.
–1
Below Expected
Below baseline. Drive-limiting decision or execution. Missed a better option or failed to execute correctly.
–2
Poor
Significantly below expectation. Rare — use sparingly. Turnover-worthy decision or execution breakdown.
